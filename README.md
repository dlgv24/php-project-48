## About project
This is a CLI application that displays file differences in various formats
## Prerequisites
- Linux, Macos, WSL
- PHP >= 8.0
- Composer >= 2.8.0
- Make
- Git
## Setup
```
$ git clone git@github.com:dlgv24/php-project-48.git

$ cd php-project-48 

$ make install  
```
